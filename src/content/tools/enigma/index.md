Text editor built for the agentic age. Inspired by Sublime Text and Tmux.

![Text editor built for the agentic age.](./enigma-screenshot-1.png)

## Features

- Lightweight & fast.
- Tabs with multiple panes. Think like Tmux but GUI.
- Built-in Vim mode.
- Agentic usage focused.

![settings is a json file](./enigma-screenshot-2.png)
